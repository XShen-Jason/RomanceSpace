🧩 各模板参数字典

本文档列出了现有可用模板以及其对应的定制参数 (data 字段内含内容)。每个模板在不传 data 时均有默认展示效果。

💌 模板一：表白信 (love_letter)

适用场景: 情书展示或表白单页面。

字段 类型 默认值 说明
title String "My Love Letter" 网页标题
sender String "Secret Admirer" 写信人
receiver String "You" 收信人
paragraphs Array ["I have liked you for a long time."] 正文字符串数组，每个元素为一个段落 
{
    "type": "love_letter",
    "data": {
        "title": "给最爱的红红",
        "sender": "小明",
        "receiver": "小红",
        "paragraphs": [
            "这是我遇见你的第100天。",
            "感谢命运让我们相视一笑。"
        ]
    }
}

🗓️ 模板二：纪念日 (anniversary)

适用场景: 在一起时光正计时或重要日期纪念。

字段 类型 默认值 说明
title String "Our Anniversary" 大标题
startDate String "2023-01-01" 起始时间日期 (格式 YYYY-MM-DD)
names Array ["Alice", "Bob"] 双方姓名数组（限2个元素）
message String "Thank you for everything." 底部宣言 
{
    "type": "anniversary",
    "data": {
        "title": "小明与小红的纪念日",
        "startDate": "2021-05-20",
        "names": ["Xiaoming", "Xiaohong"],
        "message": "一直在一起吧。"
    }
}
（新模板参数将持续补充至此列表）
